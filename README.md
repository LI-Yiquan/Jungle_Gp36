### User Manual 
#### in welcome page
input "Manual": check user manual.  
input "Game": start Jungle.  
input "Quit": quit the system.  

#### in game page
input "Piece direction": move Piece forward to the direction.  
direction: w a s d  
eg. "Cat w": move cat to the upper land.
